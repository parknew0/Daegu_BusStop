개요:
대구 시청에서 근무하다 '지도 상에 대구 버스정류장 위치표기'를 업무를 할당받았고, 이를 HTML과 네이버지도 관련 API를 활용해 구현함


전처리 진행과정:
제공받은 파일은 액셀 파일 하나로, 버스정류장 세부사항이 기재된 총 8개 행정구역(ex. 중구, 서구, 달서구 ..)의 시트로 구성되었음

구글 코랩 환경에서 해당 파일의 '버스정류장 이름'과 '해당 정류장의 주소' 열을 추출한 뒤, 네이버 지오코드 API를 활용해 위도와 경도로 변환했음
![20240825_183333](https://github.com/user-attachments/assets/aa04a673-ea44-468d-8b74-6cbb609cec99)


동작 설명:
각 행정구역 버튼을 누르면 행정구역에 속한 버스정류장의 위치를 지도상에 보여줌
'모니터링'버튼은 버스정류장 중에서도 특히 관심을 두고 있는 정류장만을 보여주는 버튼임
'스마트쉘터'버튼은 스마트쉘터가 설치된 버스정류장을 보여줌


개발 고려사항:
시청직원분들은 CS지식이 전무하기 때문에 최대한 파일구성을 쉽게 만들어서 제공해야했음.
따라서 HTML 파일 하나에 모든 소스코드를 집어넣었고, 하드코딩을 진행했음
